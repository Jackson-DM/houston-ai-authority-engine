# Commands

Executable playbooks for running the Houston AI Authority Engine consistently.
Each command is a self-contained prompt or checklist that can be executed by Jackson, Leon,
or a swarm agent without referencing the full documentation.

---

## Available Commands

| Command | Description |
|---|---|
| `run-phase-2-week.md` | Run the Phase 2 activation loop for one week using the standard 3-post cadence. |
| `run-phase-3-swarm-week.md` | Run a Phase 3 swarm-assisted weekly cycle using the Jackson Hub orchestration model. |
