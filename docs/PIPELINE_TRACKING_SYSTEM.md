# PIPELINE TRACKING SYSTEM
## Executive Conversion Monitoring

Purpose: Track executive flow from visibility → retainer. This system must answer weekly:
- Who moved forward?
- Who stalled?
- Why?

---

# FUNNEL STAGES
**Stage 0: Visibility**
- Attended Houston AI event
- Engaged on LinkedIn
- Referred

**Stage 1: Identified Executive**
- Tagged as VP / Director / C-suite

**Stage 2: Follow-Up Sent**
- Executive AI Brief delivered
- Office Hours invite sent

**Stage 3: Office Hours Attendee**
- Participated in roundtable

**Stage 4: Sprint Conversation**
- Discovery call completed

**Stage 5: Paid Sprint**
- Engagement signed

**Stage 6: Retainer**
- 90-day contract active

---

# TRACKING FIELDS
For each executive:
- Name
- Company
- Industry
- Funnel Stage
- Source (Event / LinkedIn / Referral)
- Date entered system
- Last contact date
- Next action
- Probability (%)
- Estimated contract value
- Notes

---

# WEEKLY PIPELINE REVIEW (Friday Ritual)
Review:
- # new executives identified
- # follow-ups sent
- # Office Hours attendees
- # discovery calls
- # sprints closed
- # retainers signed

Log summary in `/logs/sessions`.

---

# HEALTH METRICS
Healthy Funnel Indicators:
- 5–10 executives in Stage 2 at any time
- 3–5 executives in Stage 3 monthly
- 1–2 sprint discussions active
- 1 retainer in pipeline

*If these numbers drop → visibility or follow-up is weak.*
